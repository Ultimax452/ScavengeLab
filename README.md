**ScavengeLab** je data-driven asistent pro hru Tribal Wars, který optimalizuje sběr surovin na základě reálných herních dat.

---

## 🧠 Co to dělá

ScavengeLab není jen jednoduchý script — je to systém, který:

* vytváří optimalizované plány sběru
* porovnává plán vs realitu
* sbírá dataset z běhů
* postupně zpřesňuje výpočty

---

## ⚙️ Funkce

* 📊 Plánování sběru (efektivní rozdělení jednotek)
* 🔄 Porovnání plán vs realita
* 🧠 Dataset a jeho analýza
* 📈 Sledování přesnosti
* 🌐 Volitelné anonymní sdílení dat (opt-in)

---

## 🔒 Soukromí

ScavengeLab může (volitelně) odesílat anonymní herní data za účelem zlepšení výpočtů.

Nikdy se NEodesílají:

* jméno hráče
* ID vesnice
* souřadnice
* žádné osobní nebo identifikovatelné údaje

Sdílení dat je **ve výchozím stavu vypnuté**.

---

## 🚀 Instalace

1. Nainstaluj Tampermonkey (nebo jiný userscript manager)
2. Nahraj `scavenge-assistant.user.js`
3. Otevři Tribal Wars
4. Používej panel ScavengeLab

---

## 🧩 Struktura projektu

* `userscript/` – hlavní script asistenta
* `backend/` – Cloudflare Worker pro sběr dat
* `data/` – ukázkový dataset
